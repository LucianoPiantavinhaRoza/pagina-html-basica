H1 a H6 voltar

A tag h1 é uma das seis tags HTML de cabeçalho que representam diferentes níveis de títulos de seção. Esses níveis variam em importância e estruturam o conteúdo de uma página web.

  1. Nível mais alto: A tag h1 é o nível de seção mais alto. Ela é usada para definir o título principal de uma página ou seção.
  2. Hierarquia: As tags h2 a h6 representam níveis de títulos subsequentes, com h2 sendo menos importante que h1, h3 menos importante que h2, e assim por diante.
  3. Estruturação: O uso correto das tags de cabeçalho ajuda a estruturar o conteúdo da página, tornando-o mais legível e organizado.
  4. SEO: Além disso, as tags de cabeçalho têm implicações de SEO. Os motores de busca consideram o texto dentro das tags h1 como relevante para o tópico da página.
É importante não pular níveis de cabeçalho e começar com h1, seguido por h2 e assim por diante. Além disso, evite usar níveis menores apenas para ajustar o tamanho da fonte; em vez disso, utilize propriedades CSS para isso 1.


P voltar

A tag p é utilizada para demarcar um págrafo em HTML.

  1. Definição: A tag p (ou Elemento HTML Parágrafo) representa um parágrafo de texto. Parágrafos são, geralmente, representados visualmente como blocos de texto separados por espaços brancos verticais e/ou recuo de primeira linha.
  2. Estruturação de Texto: O uso correto da tag p ajuda a estruturar o conteúdo de uma página, tornando-o mais legível e organizado.
  3. Funcionamento:: Browsers automaticamente adicionam uma linha em branco antes e depois de cada elemento p.
  4. Estilização com CSS: Você pode usar CSS para estilizar parágrafos, alterando cor, alinhamento, recuo e outras propriedades.
Os parágrafos são usados para agrupar conteúdos relacionados, como imagens e campos de formulário, e são parte essencial da estruturação do texto em um documento HTML 1.


MARK voltar

A tag mark é utilizada para destacar um trecho de texto em HTML. Ela é frequentemente utilizada para indicar informações importantes ou para dar um estilo visual diferente ao texto.

  1. Finalidade: O elemento mark representa um trecho de destaque em um texto. Por exemplo, pode ser usado para marcar palavras relevantes em resultados de busca, referências ou partes de especial relevância para o usuário.
  2. Estilização Padrão: A maioria dos navegadores renderiza o texto dentro das tags mark com texto preto sobre fundo amarelo.
  3. Diferença ente mark e strong:: Browsers automaticamente adicionam uma linha em branco antes e depois de cada elemento p.
A tag mark não deve ser confundido com o elemento strong. Enquanto o strong denota trechos de importância especial, o mark é utilizado para indicar relevância especial. É mais apropriado para situações em que você deseja enfatizar partes do conteúdo que não foram originalmente destacadas 1.


SMALL voltar

A tag small é usada para exibir partes de um texto em um tamanho menor do que o texto circundante. Ela é frequentemente utilizada para indicar itens de importância secundária, como avisos de direitos autorais, comentários laterais e notificações legais.

O small é mais apropriado para situações em que você deseja enfatizar partes do conteúdo que não foram originalmente destacadas. 1.


I voltar

A tag i representa uma parte do texto que é destacada do restante por algum motivo, como termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo “itálico”.

A tag i não deve ser confundida com o elemento em. Enquanto o em indica ênfase ou estresse, o i é usado para destacar partes do conteúdo sem alterar a semântica original. Portanto, use o i somente quando não existir outro elemento mais apropriado para a situação. 1.


U voltar

A tag u representa um trecho de texto em linha que deve ser renderizado de forma a indicar que possui uma anotação não textual. Por padrão, esse trecho é exibido com uma linha de sublinhado simples, mas essa aparência pode ser alterada usando CSS.

A tag u não deve ser confundida com um hiperlink. Enquanto o u é usado para destacar partes do conteúdo, o hiperlink é usado para navegação. 1.


STRONG voltar

A tag strong dá ao texto uma forte importância e é tipicamente mostrado em negrito. Ele é usado para destacar partes do conteúdo que são de “grande importância”, como alertas ou sentenças cruciais.

  1. Significado Semântico: O strong fornece significado semântico ao conteúdo,indicando sua importância tanto para os navegadores quanto para os desenvolvedores.
  2. Estilização:Por padrão, o texto dentro do strong é exibido em negrito. No entanto, não deve ser usado apenas para aplicar estilo negrito. Para isso, é mais apropriado usar CSS e a propriedade font-weight.
  3. Diferença entre strong e b: O uso correto das tags de cabeçalho ajuda a estruturar o conteúdo da página, tornando-o mais legível e organizado.
    • strong: Indica grande importância e é usado para partes críticas do conteúdo.
    • b: Usado para chamar a atenção sem aumentar a importância.
O strong é um estado lógico, enquanto o negrito b é um estado físico. O primeiro separa a apresentação do conteúdo, permitindo diferentes formas de expressão, enquanto o segundo está diretamente relacionado à aparência visual 1.


OL voltar

A tag ol representa uma lista de itens ordenados. Esses itens ordenados em uma lista são mostrados com uma contagem que os precede, podendo ser de qualquer tipo, como numerais, letras, algarismos romanos ou simples símbolos.

  1. Significado Semântico: O ol fornece significado semântico ao conteúdo, indicando sua importância e ordem. A ordem é significativa, diferentemente do elemento ul que não possui ordem específica.
  2. Estilização:Por padrão, os itens dentro do ol são numerados. A aparência da numeração é definida na folha de estilos CSS associada, pela propriedade list-style-type. Não há limitação para a profundidade e a imbricação das listas definidas com os elementos ol e ul.
  3. Atributos:
    • start: Especifica o valor inicial para a numeração dos itens da lista. O tipo de ordenação pode ser com algarismos romanos, letras minúsculas, maiúsculas ou números (padrão).
    • type: Indica o tipo de numeração. Os valores possíveis são ‘a’ (letras minúsculas), ‘A’ (letras maiúsculas),‘i’ (algarismos romanos minúsculos) e ‘I’(algarismos romanos maiúsculos).
o ol é usado quando a ordem dos itens é significativa. 1.


UL voltar

A tag ul representa uma lista de itens sem ordem rígida, ou seja, uma coleção de itens que não segue uma ordenação numérica específica e cujas posições na lista são irrelevantes. Caracteristicamente, os itens em uma lista desordenada são exibidos com um marcador que pode ter várias formas, como um ponto, um círculo ou um quadrado. O tipo de marcador não é definido diretamente no HTML da página, mas sim na CSS associada, utilizando a propriedade list-style-type.

  1. Significado Semântico: O ul fornece significado semântico ao conteúdo, indicando que a ordem dos itens não é relevante. Diferentemente do elemento ol, que possui uma ordem específica, o ul não impõe nenhuma sequência particular.
  2. Estilização:Por padrão, os itens dentro do ul são marcados com pontos. A aparência desses marcadores pode ser personalizada usando CSS.
O ul é usado quando a ordem dos itens não é significativa. 1.


LI voltar

A tag li é usado para representar um item que faz parte de uma lista. Esse item deve estar contido em um elemento pai, como uma lista ordenada ol, uma lista não ordenada ul, ou um menu menu. O li representa uma única entidade dessa lista.

  1. Significado Semântico: O li fornece significado semântico ao conteúdo, indicando sua importância e ordem. A ordem é significativa, diferentemente do elemento ul, que não possui ordem específica.
  2. Estilização:Por padrão, os itens dentro do li são numerados (em listas ordenadas) ou mostrados com marcadores (em listas não ordenadas). A aparência desses marcadores pode ser personalizada usando CSS.
  3. Atributos:
    • value: Este atributo de número inteiro indica o valor ordinal atual do item na lista, definido pelo elemento ol. O único valor possível para este atributo é um número, ainda que a lista seja exibida com algarismos romanos ou letras. O atributo value não tem significado para listas desordenadas ul, nem para menus menu.
    • type: Indicava o tipo de numeração (letras minúsculas, letras maiúsculas, algarismos romanos minúsculos ou maiúsculos, números). No entanto, é recomendado usar a propriedade CSS list-style-type para estilizar os marcadores.
O li é usado quando a ordem dos itens é significativa. 1.


A voltar

A tag a é usado para criar hiperlinks em páginas da web, arquivos, endereços de e-mail, ligações na mesma página ou qualquer outro tipo de URL. O atributo mais importante do elemento a é o href, que indica o destino do link. Aqui estão alguns detalhes sobre o uso da tag a.

  1. Significado Semântico: O a fornece significado semântico ao conteúdo, indicando sua importância e ordem. A ordem é significativa, diferentemente do elemento ul, que não possui ordem específica.
  2. Estilização:Por padrão, os links aparecem da seguinte forma em todos os navegadores:
    • Um link não visitado é sublinhado e azul.
    • Um link visitado é sublinhado e roxo.
  3. Atributos:
    • href: A URL para a qual o hiperlink aponta. Os links não se restringem apenas a URLs baseadas no protocolo HTTP; eles podem usar qualquer tipo de URL suportado pelos navegadores, incluindo fragmentos de página, números de telefone e endereços de e-mail.
    • dowload: Leva o usuário a salvar a URL em vez de navegar até ela. Pode ser usado com ou sem um valor. Sem um valor, o navegador sugere um nome de arquivo/extensão com base em várias origens, como o cabeçalho HTTP Content-Disposition, o segmento final no caminho da URL ou o tipo de mídia.
    • target: Indica onde abrir o link (por exemplo, em uma nova aba ou na mesma janela).
    • rel: Define o relacionamento do URL vinculado com outros tipos de links.
    • hreflang: Sugere o idioma humano do URL vinculado.
    • ping: Uma lista de URLs separados por espaços. Quando o link é seguido, o navegador envia requisições POST com o corpo PING para esses URLs, geralmente para fins de rastreamento.
    • referrerpolicy: Controla quanto do referenciador deve ser enviado ao acessar o link. E outros atributos globais.
O a é usado para criar links entre páginas e recursos na web 1.


HR voltar

A tag hr representa uma quebra temática entre elementos de nível de parágrafo. Anteriormente, ela era usada para criar uma linha horizontal na página, mas agora seu significado é definido em termos semânticos, não apenas de apresentação.

  1. Descrição: O hr é usado para separar o conteúdo ou indicar uma mudança em uma página HTML.
  2. Uso Anterior: Nas versões anteriores do HTML, o hr representava uma linha horizontal.
  3. Uso Atual: Agora, ele é usado para criar uma quebra temática, como uma mudança de cena em uma história ou uma transição para uma nova seção.
  4. Atributos:
    • align: Define a regra de alinhamento na página (o valor padrão é “left”).
    • color: Define a cor através do nome ou valor hexadecimal.
    • noshade: Indica que não deve haver sombreamento.
    • size: Define a altura em pixels.
    • width: Define o comprimento em pixels ou porcentagem.
Para alterar a aparência da linha ou as lacunas entre ela e os parágrafos, você pode usar folhas de estilo em cascata (CSS) 1.


SUB voltar

A tag sub é usada para subscrito, ou seja, para representar texto que aparece meio caractere abaixo da linha normal. Geralmente, o texto subscrito é renderizado com uma fonte ligeiramente menor.

  1. Descrição: O sub define texto subscrito.
  2. Uso: O texto subscrito é frequentemente usado para fórmulas químicas, como H2O.
Lembre-se de que você também pode usar a tag sub para definir texto sobrescrito (como expoentes) 1.


SUP voltar

A tag sup é usada para definir texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte ligeiramente menor.

  1. Descrição: O sup define texto sobrescrito.
  2. Uso: O texto sobrescrito é frequentemente usado para fórmulas químicas, como H2O.
Lembre-se de que você também pode usar a tag sup para definir texto subscrito (como em notas de rodapé) 1.


BLOCKQUOTE voltar

A tag blockquote é usada para exibir citações longas que são extraídas de outras fontes. Anteriormente, essa tag era principalmente usada para criar uma linha horizontal na página, mas seu significado agora é definido em termos semânticos, não apenas de apresentação.

  1. Descrição: O blockquote especifica uma seção que é citada de outra fonte.
  2. Uso Anterior: Nas versões anteriores do HTML, o blockquote representava uma linha horizontal.
  3. Uso Atual: Agora, ele é usado para criar uma quebra temática, como uma mudança de cena em uma história ou uma transição para uma nova seção.
  4. Atributos:
    • cite: Especifica a fonte da citação.
Lembre-se de que, para alterar a aparência da citação, você pode usar folhas de estilo em cascata (CSS) 1.


FONT voltar

A tag font era usada para modificar o tipo de texto, tamanho e cor. Ela permitia personalizar o estilo das fontes em um site. No entanto, é importante observar que o uso da tag font e seus atributos está descontinuado no HTML5. Atualmente, a formatação de fonte, cor e tamanho é feita através do CSS.

    Atributos:
    • face: Define o tipo de fonte a ser usado.
    • size: Define o tamanho da fonte. Os valores podem ser numéricos (1 a 7) ou relativos (exemplo: +1, -2).
    • color: Define a cor da fonte. Pode ser especificada por nome (exemplo: “red”) ou valor hexadecimal (exemplo: “#FF0000”).
Lembre-se de que, para criar estilos mais avançados, é recomendável usar o CSS em vez da tag font 1.


DEL voltar

A tag del também conhecida como Elemento HTML de Texto Excluído, representa uma parte do texto que foi removida de um documento. Embora não seja necessariamente renderizada pelos navegadores com uma linha entre o texto, seu propósito é indicar que o conteúdo foi excluído.

  1. Descrição: O del é usado para marcar o texto excluído.
  2. Uso Anterior: Nas versões anteriores do HTML, o del era usado para criar uma linha horizontal.
  3. Uso Atual: Agora, ele é usado semanticamente para indicar mudanças no conteúdo ou estrutura de uma página.
  4. Atributos:
    • cite: Especifica uma URI para uma fonte que explica a alteração (por exemplo, atas de reunião).
    • datetime: Indica a data e hora da alteração.
Lembre-se de que, para estilos mais avançados, é recomendável usar CSS em vez da tag 1.


ABBR voltar

A tag abbr representa uma abreviação e opcionalmente fornece uma descrição completa para ela. Se presente, o atributo title deve conter a descrição completa e apenas ela. Essa tag é útil para fornecer informações aos navegadores, leitores de tela, buscadores e verificadores de ortografia.

  1. Descrição: A tag abbr é usado para marcar uma abreviação.
  2. Uso Atual: Ele é semanticamente usado para indicar mudanças no conteúdo ou estrutura de uma página.
  3. Atributos:
    • title: Define a descrição completa da abreviação.
Lembre-se de que, para estilos mais avançados, é recomendável usar CSS em vez da tag abbr 1.
